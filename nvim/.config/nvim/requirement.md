# Requiremenets
If not preinstalled, then install python, gcc.
Install nodejs (node, npm), python-pip

For best experience install nerd fonts for icon support, kitty terminal (or any
that supports images in terminal), luarocks package manager, ascii-image-converter, lazygit, fd, ripgrep.
For linting install python 'pip install ruff', and for JavaScript 'npm install -g @biomejs/biome'

For treesitter.nvim:
In neovim
(npm install -g tree-sitter-cli or via package manager [probably preferred]) maybe not needed.
